1 ## Command Line Evaluation program (cli)
2 
3 ### Introduction
4 
5 The `cli` program provides an easy way to fire up the priority buffer.
6 
7 It consists of a very simple state-machine that allows the user certain instructions to manipulate the buffer.
8 
9 The `cli` may be built by running `gcc cli/cli.c src/priority_buffer.c -o./cli/out && cli/out`
10 
11 Alternatively a pre-built binary is available in the `cli` directory.
12 
13 ### Menu
14 
15 The user is presented with a simple menu and the following commands are made available:
16 
17 ```
18 Enter one of the following:
19  E : Enter Values
20  P : Priority
21  G : Get Value
22  R : Reset Buffer
23  S : Show Size
24  Q : Quit
25 ```
