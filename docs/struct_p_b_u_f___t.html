Detailed Description

The pbuf_t structure holds the relevant data required for operating a single buffer. Its size is determined at compile time and depends upon the configuration applied. There is a single 8-bit tail, an 8-bit head for each priority, and an 8-bit activity byte for storing an activity flag per priority. In addition there is the buffer itself, each cell containing an element of data storage and an 8-bit pointer to the following cell.
